## Soda
Python scripts for augmenting Security Onion. 

**How to Copy Multiple Files to Security Onion Using SSH and SCP**
```bash
scp -r soda/ victor@192.168.1.69:~
```

**How to Enable SSH on a Cisco Router**
```
config t
line vty 0 4
login local
transport input ssh
```
